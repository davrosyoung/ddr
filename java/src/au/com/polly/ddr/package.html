Reduces the number of lines of data required to represent the flow rate data from a gas/oil well's production data. Key classes are;

  • {@link ExcelDataConverter} - hides the messiness involved in working with dates stored in Microsoft Excel.
  • {@link ExcelStandardizedGasWellDataExtractor} - extracts data sets from an excel spreadsheet in a standarized format.
  • {@link ExcellCellLocation} - enables us to navigate around an excel worksheet.
  • {@link GasWellDataLocator} - Whereabouts within a source of data that the data for a specific gas well may be located.
  • {@link GasWellDataEntry} - flow rates for a single time period for a specific gas well.
  • {@link GasWellDataSet} - the collection of gas well data entries for a continuous period of time for a specific gas well.
  • {@link SimpleDiscontinuityDataRateReducer} - calculates a reduced set of time intervals to represent a set of gas well data entries. Especially designed to pick up gas well transitioning from production to maintenance.
  • {@link SimpleOverlayPlotGrapher} - plots both original data set and reduced data set upon same graph.
